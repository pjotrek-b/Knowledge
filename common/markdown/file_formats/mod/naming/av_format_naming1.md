
# Format Naming

Triplet notation greatly helps reducing confusion:

  * **H.264** / **AAC** in **MP4**
  * **FFV1** / **PCM** in **MKV** (Matroska)
  * **ProRes** / **PCM** in **MOV**
  * **DPX** / **WAV** (PCM) in **a folder**
  * etc
