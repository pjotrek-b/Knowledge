Detailed Description

Subclass of the default Laminas gettext translation loader

The change performed is that the declared plural rule for the gettext file is discarded, if the file contained no plural translations.

Plural mismatches between translation files lead to an immediate exception when Laminas tries to merge them. By discarding the plurals, this situation will only arise if there's a mismatch between two actually-plural-using modules, excluding the majority that don't use plural translations.
