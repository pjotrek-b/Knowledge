Omeka\DataType\Literal::convert ( Value  $valueObject,
string  $dataTypeTarget 
)

Convert a value to the target data type.

Return true if the conversion was succesfully done. Return false if the conversion was not possible. Conversions should avoid data loss by not overwriting existing data. The value hydrator will log when conversions are not possible, and it will change the data type for you, so there's no need to do it here.
