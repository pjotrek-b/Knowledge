◆ setDefaultLocale()

Omeka\Mvc\MvcListeners::setDefaultLocale (   $locale)

Set the default locale.

Note that we only set a defualt locale when the intl extension is loaded and when the locale isn't "debug". The "debug" locale is a special case used by Omeka to detect translated strings on the page wihout switching to another language. Otherwise there will be a "Found unconstructed IntlDateFormatter" error.
