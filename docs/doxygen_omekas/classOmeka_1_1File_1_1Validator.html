?ErrorStore  $errorStore = null

Validate a file.

Validates a file against the media type and extension whitelists. Prior to calling this method the file must be saved to TempFile::$tempPath and the file's original filename must be saved to TempFile::$sourceName.

Pass the $errorStore object if an error should raise an API validation error.
