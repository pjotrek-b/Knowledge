Omeka\Controller\Admin\ResourceTemplateController::flagValid ( array  $import)
protected

Flag members and data types as valid.

All members start as invalid until we determine whether the corresponding vocabulary and member exists in this installation. All data types start as "Default" (i.e. none declared) until we determine whether they match the native types (literal, uri, resource).

We flag a valid vocabulary by adding [vocabulary_prefix] to the member; a valid class by adding [o:id]; and a valid property by adding [o:property][o:id]. We flag a valid data type by adding [o:data_type] to the property. By design, the API will only hydrate members and data types that are flagged as valid.
