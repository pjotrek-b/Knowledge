Get the browse configuration.

The browse configuration is an array containing two keys: [ 'sort_by' => '<sort_by_param>', 'sort_order' => '<sort_order_param>', ]

Note that context determines the origin of user-configured data: "public" context derives from site settings; "admin" context derives from user settings.
